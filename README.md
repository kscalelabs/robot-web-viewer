## robot-web-viewer
URDF Viewer using Reactjs, [three.js](https://threejs.org/) and [urdf-loader](https://github.com/gkjohnson/urdf-loaders)


### setup

```
$ git clone https://github.com/nakano16180/robot-web-viewer.git
$ cd robot-web-viewer/
$ yarn install
```

### run

```
$ yarn start
```

### deploy

```
$ yarn deploy
```
